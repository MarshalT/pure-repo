# Pure Repo

一个追求纯粹的代码仓库，专注于代码质量和最佳实践。

## 项目理念

- **简单**：保持代码简单直接，避免不必要的复杂性
- **纯粹**：专注于核心功能，避免冗余代码
- **可维护**：遵循代码规范，保持良好的代码组织
- **可测试**：重视单元测试，保证代码质量

## 项目结构

```
pure-repo/
├── src/           # 源代码目录
├── tests/         # 测试文件目录
├── docs/          # 文档目录
└── examples/      # 示例代码目录
```

## 开发规范

### 代码风格
- 使用 ESLint 进行代码检查
- 遵循 Airbnb JavaScript 风格指南
- 使用 Prettier 进行代码格式化

### 提交规范
- feat: 新功能
- fix: 修复问题
- docs: 文档修改
- style: 代码格式修改
- refactor: 代码重构
- test: 测试用例修改
- chore: 其他修改

## 开发流程

1. Fork 项目
2. 创建特性分支
3. 提交代码
4. 发起 Pull Request
